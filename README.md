# Project Analysis Tool with DB

A powerful tool designed to analyze projects that include database structures. It provides developers with flexible options to inspect and compare database schemas and stored procedures.

---

## 📋 Features / المميزات

This tool allows you to analyze your database project in three distinct ways:

### 1. Direct Database Connection (الاتصال المباشر)

* **EN:** Connect directly to your SQL Server to analyze stored procedures and schema definitions in real-time.
* **AR:** الاتصال المباشر بخادم SQL لتحليل الإجراءات المخزنة (Stored Procedures) وهيكلية قاعدة البيانات في الوقت الفعلي.

### 2. Backup File Analysis (تحليل ملف النسخة الاحتياطية)

* **EN:** Analyze the project structure by importing a standard database `.bak` file.
* **AR:** تحليل المشروع من خلال استيراد ملف النسخة الاحتياطية `.bak` لقاعدة البيانات.

### 3. Merge & Compare Backup Files (دمج ومقارنة ملفات Backup)

* **EN:** Merge two `.bak` files to identify differences and compare stored procedures between versions.
* **AR:** دمج ملفين `.bak` لمقارنة الإجراءات المخزنة والتعرف على الفروقات بين نسختين من قاعدة البيانات.
